# WBest

Fast and free online tools available at [wbest.app](https://wbest.app/). No sign-up, no uploads, no tracking.

WBest is a suite of single-purpose web tools that run entirely in the browser. Since all processing is done client-side, your data never leaves your device. Each tool is built as an independent, lightweight web app without external server dependencies.

## Key Features

- **Privacy-first**: All processing is local. Your data is never uploaded, logged, or shared.
- **Offline capability**: Once loaded, every utility works fully without an active internet connection.
- **No accounts or sign-up**: Zero friction access with no ads, tracking, or limitations.
- **Zero build step**: Written in vanilla HTML, CSS, and JavaScript.
- **Responsive design**: Modern, lightweight, and responsive interfaces with automatic light/dark mode support.

## Running the Tools

Because these tools are built with vanilla web technologies, they require no local servers, build pipelines, or packages.

To run a tool locally:
- **Clone this repository**: Open the `index.html` file inside any tool's directory directly in your browser.
- **Save from the live site**: Right-click and save the page of any tool from [wbest.app](https://wbest.app/), then open the saved HTML file locally.

No installation, build tools, or active internet connection is required.
