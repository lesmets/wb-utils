In deinem Browser erstellt

Das Paket wird vollständig auf deinem Gerät zusammengestellt. Dein Firmenname, deine Markenfarbe und dein Logo werden nirgendwo hochgeladen, protokolliert oder gespeichert.

Wirklich in sich geschlossen

Jedes Tool ist eine einzelne HTML-Datei ohne Abhängigkeiten, ohne Build-Schritt und ohne Netzwerkanfragen. Hoste den Ordner überall, wo ein Browser ihn erreicht.

Deine Marke, ein Klick

Jede Seite trägt deinen Namen in Titel und Kopfbereich, deine Farbe in Light- und Dark-Mode und dein Logo, damit sich die Tools wie deine eigenen anfühlen.

So funktioniert es

Gib deinen Team- oder Firmennamen ein, wähle eine Markenfarbe und füge optional ein Logo hinzu. Wähle dann die Tools aus, die dein Team wirklich nutzt, und die Sprachen, die du ausliefern willst. Die Vorschau zeigt genau, wie eine paketierte Seite aussehen wird. Beim Klick auf Erstellen lädt der Packager eine neutrale, reduzierte Vorlage jedes gewählten Tools, wendet dein Branding an und stellt alles zu einem einzigen ZIP zusammen, komplett lokal in deinem Browser.

Das ZIP enthält einen Ordner pro Tool, jeweils mit einer in sich geschlossenen index.html samt Übersetzungen und Favicons, sowie eine Startseite ganz oben, die die enthaltenen Tools verlinkt. Es gibt nichts zu installieren und nichts zu konfigurieren: Entpacke es auf einen beliebigen statischen Webserver, Intranet-Host oder eine Dateifreigabe und es funktioniert. Da jede Seite vollständig im Browser läuft und keine Netzwerkanfragen stellt, eignet sich das Paket gut für Umgebungen, in denen Daten das Haus nicht verlassen dürfen.

Die paketierten Seiten sind dieselben Tools, die auf wbest.app laufen, minus allem, was nur online Sinn ergibt: keine Analyse, kein Suchmaschinen-Markup, keine Artikeltexte, nur das Tool. Jede Seite behält eine einzeilige Danksagung an WBest Tools im Fußbereich.

Häufig gestellte Fragen

Was enthält das Paket?

Einen Ordner pro Tool mit einer einzelnen, in sich geschlossenen index.html (plus Übersetzungen und Favicons), eine Startseite, die die gewählten Tools verlinkt, und dein Logo, falls du eines hinzugefügt hast. Die Seiten sind dieselben Tools, die auf wbest.app laufen, reduziert auf das Tool selbst und umbenannt mit deinem Namen und deiner Farbe.

Wie hoste ich das Paket?

Entpacke es überall dorthin, wo ein Browser es erreichen kann: auf einen beliebigen statischen Webserver, einen Intranet-Host, eine Firmen-Dateifreigabe oder sogar einen USB-Stick. Es gibt keinen Build-Schritt, keine Datenbank und keine Konfiguration. Auch das direkte Öffnen der index.html von der Festplatte funktioniert.

Werden mein Logo oder mein Firmenname irgendwohin hochgeladen?

Nein. Der Packager läuft vollständig in deinem Browser: Er lädt die neutralen Tool-Vorlagen, wendet dein Branding lokal an und erstellt das ZIP auf deinem Gerät. Dein Name, deine Farbe und dein Logo verlassen nie deinen Browser.

Brauchen die paketierten Tools eine Internetverbindung?

Nein. Jede paketierte Seite ist vollständig in sich geschlossen, stellt keine Netzwerkanfragen und funktioniert auch offline. Genau das macht sie sicher für sensible Daten in einem internen Netzwerk.

Darf ich diese Tools mit eigenem Branding selbst hosten?

Ja, genau dafür ist dieser Packager da. Jede erzeugte Seite behält eine kleine Zeile „Powered by WBest Tools“ im Fußbereich, die auf wbest.app verlinkt. Bitte lass diesen Hinweis stehen.
